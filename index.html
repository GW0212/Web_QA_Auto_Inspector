    Test Items

    자동 QA 검사항목

    실제 웹 검수 업무에서 반복적으로 확인하는 항목을 기준으로 구성했습니다.

    01

    페이지 접속 검사

    URL 접속 가능 여부, 상태 코드, 초기 로딩 성공 여부를 확인합니다.

    02

    이미지/링크 검사

    깨진 이미지, alt 누락, 내부/외부 링크, 빈 href를 점검합니다.

    03

    반응형 UI 검사

    데스크톱/모바일 화면 기준으로 가로 스크롤과 레이아웃 깨짐을 확인합니다.

    04

    콘솔 에러 검사

    JavaScript 에러, 리소스 로딩 실패, 네트워크 에러를 리포트화합니다.

    05

    SEO/접근성 검사

    title, description, h1, canonical, 버튼 이름, input label을 확인합니다.

    06

    리포트 Export

    검사 결과를 JSON/HTML 파일로 내려받아 QA 산출물로 활용할 수 있습니다.

    ⌁
