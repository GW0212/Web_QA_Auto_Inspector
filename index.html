    Test Items

    자동 QA 검사항목

    실제 웹 검수 업무에서 자주 확인하는 접속, 이미지, 링크, 모바일 화면, 오류, 속도, 보안, 입력칸 안내 항목을 일반 사용자도 이해하기 쉽게 정리했습니다.

    01

    페이지 접속 검사

    입력한 주소로 페이지가 정상적으로 열리는지 확인합니다.

    02

    이미지/링크 검사

    이미지가 안 보이거나 링크가 비어 있는 문제를 확인합니다.

    03

    반응형 UI 검사

    PC와 모바일에서 화면이 삐뚤어지거나 옆으로 밀리는지 확인합니다.

    04

    콘솔 에러 검사

    페이지 내부 오류나 필요한 파일을 불러오지 못한 문제를 확인합니다.

    05

    검색/접근성 검사

    검색에 필요한 제목·설명과 버튼/입력칸 설명이 있는지 확인합니다.

    06

    체감 속도 검사

    첫 화면이 빨리 보이는지, 클릭 반응이 빠른지, 화면이 흔들리지 않는지 확인합니다.

    07

    보안 연결 검사

    안전한 주소로 접속되는지, 안전하지 않은 파일이 섞여 있지 않은지 확인합니다.

    08

    기본 보안 설정 검사

    브라우저가 페이지를 더 안전하게 열 수 있도록 필요한 기본 보안 설정을 확인합니다.

    09

    키보드/입력칸 검사

    키보드만으로 이동 가능한지, 입력칸 이름과 오류 안내가 보이는지 확인합니다.

    10

    문서 구조 검사

    페이지 언어, 모바일 화면 기준, 제목 순서가 자연스러운지 확인합니다.

    11

    새 창 링크 안전성

    새 창으로 열리는 외부 링크가 안전하게 처리되는지 확인합니다.

    12

    리포트 저장

    검사 결과를 파일로 저장해 포트폴리오나 업무 보고 자료로 활용할 수 있습니다.

    ⌁

    아직 검사 결과가 없습니다

    URL을 입력하고 QA 검사를 시작하면 결과 요약, 상세 이슈, 항목별 리포트, 다운로드 영역이 표시됩니다.
